Modifiyng Attributes

Attributes

  • it can be src , href class, id ,type,value

getAttribute

const firstInput =document.querySelector('input')
firstInput.getAttribute('type')// text

setAttribute

const firstInput =document.querySelector('input')
firstInput.setAttribute('type','email')

There are a few attributes that u can directly acces and modify as well instead of using getAttribute or setAttribute ,
one is id
// input type="text" id="firstname">
const firstInput=document.querySelector("input")
firstInput.id// first-name
firstInput.id ="full-name"--> this changes the id name
