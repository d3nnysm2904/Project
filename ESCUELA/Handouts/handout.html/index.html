Paragraphs, Headings, and Breaks Paragraphs HTML is often used to present largely-textual information. It’s common for text to appear inside a paragraph:

It was a dark and stormy night.

It’s possible for other tags to go inside a p tag, but a paragraph cannot contain another paragraph. Headers There are different heading elements you can use, from largest/most-important, to smallest/least-important: h1 (largest) to h6 (smallest) You’ll often use several of these, in order

Lists in HTML HTML can make two kinds of lists: unordered (bulleted, typically) ordered (numbered, typically) Unordered Lists A sequence of list items surrounded by an unordered list element:
  • apple
  • berry
  • cherry
apple berry cherry The list should only directly contain list items — but list items can contain text and other elements. Ordered Lists A sequence of list items surrounded by an ordered list element:
  1. first
  2. second
  3. third
first second third The list should only directly contain list items — but list items can contain text and other elements. Attributes That’s our first example of an attribute, which can be a part of a tag. Attributes provide other info to a tag, such as: Changing the appearance of the element Showing links what they link to Telling image tags where the image file can be found Attributes are always a word, followed by an equal sign, and a value (typically in quotes, like our example) Where To Link You can also link to pages deeper down at your site: About Elmo This links to the page “elmo.html” in the “employees” directory in this folder You can also link to pages higher up at your site: Another Page This links to a page one folder up than the current folder. You can also link to other sites: Adorable! This links directly to a page on Wikipedia. To link to another site, your href attribute needs to have a complete URL — typically starting with “http://” or “https://” Images HTML can show images, either from your site or another site: https://tinyurl.com/y39p455z This is another tag with an “attribute” — src is the location of the actual image file. Image Types Web browsers can show every type of image, but common types: GIF JPG PNG You can export into one of these types in most image-designing programs Entities HTML is meant to be text, and it can be tricky to get special symbols, like copyright symbols or curly quotes, into HTML HTML can have entities which are text spellings for common symbols. Entities are always an ampersand, a single word, and a semicolon: &example; Less Than / Greater Than Since HTML tags use angle brackets, if you try to use these symbols in your HTML, it will often confuse browsers. Instead use: < Turns into a “less-than” symbol: < > Turns into a “greater-than” symbol: > Copyright Symbol © Turns into a “copyright” symbol: © Curly Quotes You can put “straight quotes” directly into HTML: She said "Hello" Curly quotes can be trickier, and are often done with entities: “ Turns into a “left-double-quote”: “ ” Turns into a “right-double-quote”: ” To write a pretty quote: She said: “Hello!” Emmet Emmet is a library for writing html with fewer keystrokes We will be covering how to generate html quickly with Emmet shortcuts Try it out Emmet comes pre-installed with VScode Create a new file with a .html extension Type ! and pres tab You should see some automatically generated html like the following: // index.html Document Creating elements with Emmet A pain point when writing html tags is typing the opening and closing brackets With Emmet we just need to specify the tag type and press tab h1 tab
